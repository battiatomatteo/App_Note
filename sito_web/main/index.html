Boat Note

  • Il fine di questa app è quello di creare note e promemoria, i dati dell'utente verranno salvati in un database mentre le note ed i promemoria in un file .txt .

  • Al momento è possibile nella cartella 'screen_app' sono presenti screen di ogni sezione presente nell'app. Finita la parte grafica, tra poco verrà aggiunta la parte del salvataggio dei dati.

  • Lo scopo della creazione di questo progetto è quello di apprendere attraverso la costruzione di un'app il linguaggio Dart.
